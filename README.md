## Build
**Build with defaults**
./build.sh

**Build with specific QT location**
QT5_DIR=/opt/Qt5.10.0/5.10.0/gcc_64/lib/cmake/Qt5 ./build.sh

**Build using multiple cores**
./build.sh -j 4
